
# GSMP Season 5 Modpack
This is the _official_ and _default_ modpack for GSMP Season 5.
It contains QoL mods as well as the core mods that are _required_ to join the server.

## For Devs:

To add a mod, run:
```bash
packwiz mr add <link/slug/query>
```

To remove a mod, run:
```bash
packwiz remove <name>
```

### Note:
After manual changes run
```bash
packwiz refresh
```
